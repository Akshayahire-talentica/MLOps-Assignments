🖥️ UI Access (Requires Port-Forward)

⚠️ Important: These UIs require you to run port-forward commands in separate terminal windows FIRST.
📊 MLflow UI:
kubectl port-forward svc/mlflow 5000:5000 -n mlops
Then open: http://localhost:5000
📈 Prometheus UI:
kubectl port-forward svc/prometheus 9090:9090 -n mlops
Then open: http://localhost:9090
📖 API Documentation:
kubectl port-forward svc/mlops-api 8000:8000 -n mlops
Then open: http://localhost:8000/docs

How to run port-forward commands:

  1. Open a NEW terminal window
  2. Copy and paste one of the kubectl port-forward commands above
  3. Press Enter (it will keep running)
  4. Click the link to open the UI
  5. Leave the terminal running while using the UI
  6. Press Ctrl+C in the terminal to stop when done
